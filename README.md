# Quiz with Hidz - Interactive Web Application

An interactive, web-based quiz application designed for different age groups, built as part of the **SYBSc IT** curriculum.

## 🚀 Project Overview
This project is a dynamic quiz platform that categorizes questions based on the user's expertise or age group. It provides an engaging UI/UX to test knowledge across various subjects.

### Key Features:
* **Multi-Level Difficulty:** Categorized questions for Junior, Intermediate, and Senior levels.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Interactive UI:** Uses CSS blurs, transitions, and positioning for a modern aesthetic.
* **Subject Selection:** Users can choose specific topics to be tested on.

## 🛠️ Technologies Used
* **Frontend:** HTML5, CSS3 (Flexbox, Animations)
* **Logic:** JavaScript (DOM Manipulation, Quiz Logic)
* **Version Control:** GitHub

## 📂 Project Structure
* `index.html` - The main entry point.
* `login.html` - User authentication interface.
* `about.html` - Project and developer information.
* `style.css` - All custom styling and layout rules.
* `age.html` - Age group selection module.
* `subjects.html` - List of subjects for user to choose.
* `QuizPanel.html` - The core dashboard for the quiz experience.
